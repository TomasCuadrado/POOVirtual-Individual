# Operaciones con clases, métodos particulares
1. Defina una clase base llamada Vehículo con un método desplazar que imprima un mensaje simple, por ejemplo: “Vehículo desplazándose”.
2. Extienda esta clase en dos subclases: Coche y Bicicleta.
3. En la clase Coche, sobrecargue el método desplazar para que acepte un parámetro que indique si el desplazamiento es en modo 'automático' o 'normal', cambiando el comportamiento del método según este modo.
4. En la clase Bicicleta, sobrescríbala para reflejar un modo de desplazamiento más específico para bicicletas.
5. Finalmente, en la clase principal, cree instancias de cada subclase y demuestre la sobrecarga y sobrescritura, llamando al método desplazar adecuado para cada instancia de los objetos creados.
